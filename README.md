# Notes

Node js app to generate multiplication table automatically , sending any number

```
Options:
      --help                                Show help                  [boolean]
      --version                             Show version number        [boolean]
  -b, --base                                Is the base of multiplication table
                                                             [number] [required]
  -l, --list                                show the table on console
                                                      [boolean] [default: false]
  -h, --defines up to which number the      define
  table will be compiled                                  [number] [default: 10]
```
